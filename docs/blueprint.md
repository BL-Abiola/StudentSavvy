# **App Name**: StudentSavvy

## Core Features:

- GPA Calculation: Calculate GPA based on entered course grades and credits.
- Task Management: Create, prioritize, and track tasks with due dates.
- Study Planner: Schedule study sessions with topic, date, time, and notes.
- Class Schedule: Manage class schedules with day, time, and location.
- GPA Prediction Tool: Predict future GPA based on expected grades (AI-powered tool for premium users).
- AI Study Session Synthesizer: Generates short study sessions from pasted notes with the help of Gemini AI

## Style Guidelines:

- Primary color: Indigo (#4F46E5) for a professional feel.
- Background color: Light gray (#F9FAFB), a slightly desaturated indigo for a calm background.
- Accent color: Teal (#10B981) to represent progress and achievement.
- Body and headline font: 'Inter' for a clean, modern sans-serif.
- Use consistent, outline-style icons for navigation and task representation.
- Maintain a clear, sectioned layout with ample white space for easy content scanning.
- Use subtle animations for task completion and data updates to enhance user feedback.
